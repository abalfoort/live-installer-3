Situada sobre os ombros de gigantes

  • SolydXK é uma distribuição GNU/Linux baseada, derivada e compatível com o Debian.
  • SolydXK usa tanto o levíssimo desktop Xfce para o SolydX ou o o altamente configurável e incrivelmente atraente desktop KDE para o SolydK.
  • Nossas inovações, o rápido desenvolvimento do Debian e a quantidade enorme de pacotes Debian tornam o SolydXK um dos sistemas operacionais mais atraentes para desktop disponível para usuários domésticos.
